# Guide

## Usage Guide

### Installation

```bash
npm install celf-cli -g

# check info
ccli info

# check version
ccli -V
```

### Using Template

```bash
# show template list
ccli tpl
```

### Using Quick Command

#### Development Environment

```bash
# init commitizen
ccli quick init cz

# init tailwindcss
ccli quick init twc
```
